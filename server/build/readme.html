What's New

Open Source Javascript

We have released all the source code for our core javascript components under the GNU GPL version 3. This is the core javascript components, and does not include our back-end components. If you wish to get involved in the development of the javascript components, please visit our GoogleCode page

Improved .Net Support

We now have new ASP.NET support for all our components, and they have been totally redone. The documentation for the new components are is included with the installer. We hope to have more documentation soon.

Bug Reporting

Please log any issues with the Open Source components either on our Google Code Issue Tracker or on our Google Group, and we will do our best to address these issues. If you have a support contract, please feel free to visit our online helpdesk at nitobi.zendesk.com.

Installing Complete UI

To install Complete UI, simply run the downloaded installer. If you have a previous version of Complete UI installed, you will need to uninstall the previous version before installing the new version.

Installing Dreamweaver Extensions

To install the Dreamweaver Extensions, open the extentions package, nitobi_cui_ext.mxp, in Adobe Extension Manager. The package will be found in the tools subdirectory of the Complete UI install directory.

Directory Structure

The Complete UI install directory contains the following directories:

  • javascript_css - contains minified JavaScript code and CSS for each component.
  • server_libraries - contains server components side code for PHP, CFM, ASP, and JSP platforms.
  • docs - links to http://support.nitobi.com for full JavaScript API and HTML tag reference for all components as well as a link to the online Knowledge Base
  • samples - contains various samples for each of the components.
  • tools - contains the Dreamweaver Extension package for the Nitobi Dreamweaver Extensions.
  • build - for Enterprise license customers and the free Nitobi Ajax Toolkit the source code and build tools can be found here.
