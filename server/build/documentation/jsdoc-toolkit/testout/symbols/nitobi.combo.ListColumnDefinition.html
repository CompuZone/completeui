{String} GetImageHandlerURL()
Returns the image url of the column. If the ColumnType is IMAGE this property must be specfied. It contains the URL of a page that will serve up an image given an argument. The database column will contain image identifiers; this identifier will be used as an argument to the page, e.g. if the URL is http://localhost/ImageServer.aspx?, then combo will call it as follows http://localhost/ImageServer.aspx?Image=DataFromColumn. It will expect a binary image to be written out by the page. For example, the values in this column could be the id's of people stored in the database. When the combo is loaded the row value sends a request to the image handler, e.g. by calling http://localhost/ImageServer.aspx?Image=shiggens. ImageServer.aspx will check if shiggens is online, and, if so, send back an image specifying so.
