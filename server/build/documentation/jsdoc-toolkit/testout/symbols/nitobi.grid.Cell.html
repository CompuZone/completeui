Class nitobi.grid.Cell

The Cell object represents a single row/column combination in the Grid. It can be used to get or set values of cells in the Grid, or to manipulate the look of a particular cell.

A cell is represented in two ways: as a rendered html element and as an xml node. The Cell object provides a means to affect either representation

setValue(value, display)
This sets the value of a cell which results in the value being set in the datasource (its xml node) and the view of the data being updated (the html rendered for the cell). If the cell belongs to a number or date column, for example, the value will have the appropriate mask applied to it.

Example


function setCellValue(row, col, value)
{
	var grid = nitobi.getComponent('grid1');
	var cell = grid.getCellObject(row, col);
	// Note the second parameter is optional
	cell.setValue(value);
}

Parameters:
{String} value
The value which will be put in the database.
{String} display
The value which will be displayed in the Grid. This is optional.
