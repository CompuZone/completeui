Class nitobi.tree.Tree


Extends nitobi.ui.Element.
The nitobi tree. A tree declaration takes this form: 


<ntb:tree id="tree1" cssclass="folders" expanded="false">
	<ntb:children>
		<ntb:node label="United States">
			<ntb:children>
				<ntb:node label="New York City" ></ntb:node>
				<ntb:node label="Washington" ></ntb:node>
				<ntb:node label="Los Angeles" ></ntb:node>
			</ntb:children>
		</ntb:node>
		<ntb:node label="Canada" >
			<ntb:children>
				<ntb:node label="Toronto"></ntb:node>
				<ntb:node label="Vancouver"></ntb:node>
				<ntb:node label="Calgary"></ntb:node>
			</ntb:children>
		</ntb:node>
	</ntb:children>
</ntb:tree>
Calling nitobi.loadComponent('tree1'); will render a tree at the location of that XML declaration in your web page. To access this API you get the tree object from its node in the HTML document. We can, for example, add a parameter to the gethandler: 

var tree = $('tree1').jsObject;
var gethandler = tree.getGetHandler();
gethandler = nitobi.html.Url.addParameter(gethandler, 'testname', 'testvalue');
tree.setGetHandler(gethandler);

find(key, value)
Returns an Array of the nodes matching the given key/value pair. Typical usage would be to find a node with a known id: 

	var myNode = tree.find('id','myNodeId')[0];
Or to find all the nodes of a specific type: 

	var fooNodes = tree.find('type','foo');
Note: The running time for this method increases linearly with the number of nodes found.
