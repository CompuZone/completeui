<static> nitobi.lang.close(context, func, params)
Creates a closure for a function and a context. Also supports the passing of arguments. As an alternative to defining inline anonymous functions that alias the this keyword one can use a closure. The conventional approach to creating a closure is shown below: 

example.Account.prototype.getBalance = function() {
  var xhr = new nitobi.ajax.HttpRequest();
  var _this = this;
  xhr.onGetComplete.subscribe(function(evtArgs) {
    _this.balance = evtArgs.response;
  });
  xhr.get("balance.do");
}
An example of the same code using closures is shown below. This code is slightly more verbose yet easier to document and understand as well as being less prone to Internet Explorer memory leak patterns. 

example.Account.prototype.getBalance = function() {
  var xhr = new nitobi.ajax.HttpRequest();
  xhr.onGetComplete.subscribe(nitobi.lang.close(this, this.getBalanceComplete));
  xhr.get("balance.do");
}
example.Account.prototype.getBalanceComplete = function(evtArgs) {
  this.balance = evtArgs.response;
}
Parameters:
{Object} context
The context for the function to be executed in.
{Function} func
The function to be executed.
{Array} params
An array of arugments that can be passed to the function.
