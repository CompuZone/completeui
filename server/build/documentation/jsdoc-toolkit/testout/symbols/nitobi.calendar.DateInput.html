Method Detail
{String} getDisplayMask()
Returns the display mask for the DateInput. The display mask is used to make the date in the input more human readable. Defaults to "MMM dd yyyy", e.g. "June 24 2008".
{String} getEditMask()
Return the edit mask for the DateInput. The edit mask is applied to the value in the input field when the input if focussed. This is to make date entry easier. Will default to the value of the display mask.
