Tabstrip Options (1 of 2)
Choose general Tabstrip options. You can specify the dimensions of the Tabstrip as well as the alignment of the tabs, and how much the tabs will overlap. Align, overlap, width, height.
Id: The id is required.
The id to apply to the Tabstrip. Must be unique.
Width: Value must be expressed with 'px'.
The width of the Tabstrip component, expressed in px.
Height: Value must be expressed with 'px'.
The height of the Tabstrip component, expressed in px.
Align:
The horizontal alignment of the tabs.
Overlap: Must be a number.
The overlap between tabs.
Tabs (2 of 2)
Now add individual tabs to the Tabstrip component. Choose a tab from the list to set its properties.
Tabs:
You must add at least one tab.
Tab Properties:
A DOM tab gets its source from a DOM node on the page.
Label:
The label to apply to this Tab.
Width: Must be expressed with 'px'.
Specifies the width of this tab, expressed in px.
Source:
The source for the tab can be an external site (e.g. http://www.nitobi.com), a page on the same domain (ajax.html) or a DOM node specified by id.
