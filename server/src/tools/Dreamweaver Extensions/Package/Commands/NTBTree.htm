Tree Options (1 of 1)
Choose Tree options. This will insert a sample Tree into the document with the properties set from the options below.
Id: The id is required.
The id to apply to the Tree. Must be unique.
Css Class:
The css class specifies the theme to apply to the Tree. Choose from the list of included themes, or type the name of a custom theme into the field.
Root Enabled:
Choose true if you wish to have a special root style applied to the top root node of the tree.
Expanded:
If set to true, every node in the tree will be expanded to show its children. If set to false, every node will be collapsed. Individual nodes can override this property by setting their own expanded attributes.
Hover Highlight:
Specifies whether or not a hover class is added to nodes when they are hovered over.
