Spotlight Tour Options (1 of 2)
Specify general Spotlight tour options.
Starting Effect:
The starting effect describes how the Spotlight component will transition into the tour.
Callout Style:
The Callout style is the style applied to tour Callouts.
Lens Type:
The lens type determines the style of vignette when the tour focuses on a page element.
Lens Oversize:
The lens oversize specifies how much bigger to make the lens than the object it is focused on. The default value is 0.2.

Tour Steps (2 of 2)
Now we need to specify each Spotlight tour step. The tour will run the steps in the list from top to bottom. Click the "Add" button below to begin adding steps to this Tour.
Steps:
There must be at least one step in the tour.
Tour Step Options:
A Callout Step will focus on an element and attach a Callout to it.
Element ID:
The id of the dom element to attach the Callout to. You can choose "None" to have a Callout be unattached and centered on the page.
