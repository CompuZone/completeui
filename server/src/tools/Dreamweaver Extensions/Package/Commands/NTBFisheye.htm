Fisheye Options (1 of 2)
Choose general Fisheye options. You can specify the
Id: The id is required.
The id to apply to the Fisheye. Must be unique.
Grow Percent: Must be a number.
The maximum size to grow the Fisheye component.
Icon Width: Value must be expressed with 'px'.
The width to apply to the Menu Item icons. Expressed in px.
Open Direction:
The direction to open the Fisheye menu when it expands.
Expand Direction:
The lens oversize specifies how much bigger to make the lens than the object it is focused on. The default value is 0.2.
Fisheye Items (2 of 2)
Now you can add items to the Fisheye menu and choose each items icon from your filesystem.
Menu Items:
You must add at least one item.
Menu Item Options:
Label:
The label to apply to this Menu Item.
Image Source:
The image source is a url to an image relative to the site root.
Onclick:
The javascript code to execute when this menu item is clicked on.
