Date Input Options (2 of 3)
Define the characteristics of the Date Input.
Display Mask: Invalid mask
The mask applied to the date in the input when it is blurred. Available masking tokens are (January 30, 2008 as the example date): yy(08), yyyy(2008), M(1), MM(01), MMM(January), NNN(Jan), d(30), dd(30), EE(Wednesday), E(Wed).
Edit Mask: Invalid mask
The mask applied to the date in the input when it is focussed. Available masking tokens are (January 30, 2008 as the example date): yy(08), yyyy(2008), M(1), MM(01), MMM(January), NNN(Jan), d(30), dd(30), EE(Wednesday), E(Wed).
Width: Invalid width.
The width applied to the Input. Will by default use the css class as defined by the chosen theme.
Editable: True False
Check to make the input editable, uncheck to disable editing.
