Description

The Nitobi JSF components allow JSF developers to use Nitobi's full featured suite of AJAX components within the JSF lifecycle.

Features

  • Script and style resources loaded automatically. Each component knows what dependencies it has and it loads them straight from the jar.
  • Simpler Gets and Saves with the new listener components.
  • All attributes (other than id) support value expressions.

Who is it for?

This is an early preview release of our forthcoming JSF components and is meant for JSF developers who are interested in using our suite of full featured Ajax components. The javascript files included in nitobi-jsf.jar will expire on July 01, 2008.

Requirements

The Nitobi JSF Components are built for JSF 1.2. MyFaces is not currently supported.

Directory Structure

The Complete UI install directory contains the following directories:

  • javadoc - Generated documentation for the JSF components.
  • lib - all the necessary lib files for the JSF components.
  • NitobiJSFWeb - an eclipse project with samples of each of the JSF components. Simply import the project as an exisiting project.
  • tlddoc - Generated documentation for the JSF tag library.
